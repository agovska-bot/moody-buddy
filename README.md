# Moody Buddy+

A new kind of digital journal for kids aged 7-12. Our mission is to help kids understand their feelings and build happy habits in a fun and private way.

This project is built with React, TypeScript, and Vite. It's ready to be deployed on **Vercel**.

### Prerequisites

- Node.js (LTS version)
- A Google AI API Key

### Local Setup

1.  **Clone/Download the repository.**
2.  **Open Terminal** inside the project folder (`moody-buddy-plus`).
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Create an environment file:**
    - Create a file named `.env` in the root of the project.
    - Add your Google AI API key to it.
      ```
      API_KEY=YOUR_AI_KEY_HERE
      ```
5.  **Run the development server:**
    ```bash
    npm run dev
    ```

### Deployment on Vercel

1.  Push the code to GitHub.
2.  Go to Vercel and **Import** the project.
3.  In **Environment Variables**, add:
    - **Name:** `API_KEY`
    - **Value:** `YOUR_AI_KEY_HERE`
4.  Click **Deploy**.
